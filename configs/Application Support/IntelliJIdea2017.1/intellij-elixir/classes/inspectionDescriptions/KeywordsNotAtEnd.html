Keywords appear before the end of a list. Move keywords after positional arguments. Keywords appear before end of list or argument, so they are followed by positional arguments, which is a syntax error in Elixir. 
one.(
  one,
  two positional, key: value,
  three
)
Either surround the no parentheses expression argument with parentheses, or move the the keywords to the end of the list if it wasn't meant to be a no parentheses expression. 
one.(
  one
  two(positional, key: value),
  three
)
OR 
one.(
  one,
  two,
  three,
  key: value
)
