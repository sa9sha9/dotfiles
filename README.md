# Usage
`Usage: ./entry.sh [(deploy/init)]`

## Deploying
```bash
./entry.sh deploy
```

## Initialization
```bash
./entry.sh init
```

## Both
```bash
./entry.sh
```
# Operations
## Deploying
`ln -snfv symlink/.zshrc ~`などでドットファイルをホームディレクトリにシムリンク。

## Initialization
zshのインストールやmacOSのdefaultセッティングなどを行う。
