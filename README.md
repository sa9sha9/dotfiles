# Usage
`Usage: $0 [-e (deploy/init)]`

## Deploying
```bash
./entory.sh -e deploy
```

## Initialization
```bash
./entory.sh -e init
```

## Both
```bash
./entory.sh
```
# Operations
## Deploying
`ln -snfv symlink/.zshrc ~`などでドットファイルをホームディレクトリにシムリンク。

## Initialization
zshのインストールやmacOSのdefaultセッティングなどを行う。
